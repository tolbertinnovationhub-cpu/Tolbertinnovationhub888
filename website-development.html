Website Development

Learn practical website creation using HTML, CSS, and JavaScript.

Course Structure

  1. Introduction — tools and setup
  2. HTML basics — structure and semantic tags
  3. CSS fundamentals — layout and responsive design
  4. JavaScript essentials — interactivity and DOM
  5. Project — build and deploy a simple website
  6. Assessment — project review and certificate

Lessons & Materials

Lesson 1 — Tools & Setup

Install a code editor, set up a local development folder, and preview HTML pages in the browser.

Lesson 2 — HTML Basics

Learn page structure, headings, paragraphs, links, lists, and images.

Lesson 3 — CSS Fundamentals

Style pages using selectors, box model, flexbox, and responsive breakpoints.

Lesson 4 — JavaScript Essentials

Add interactivity: event listeners, DOM manipulation, and simple form handling.

Project

Build a 3-page website (home, about, contact) and deploy it using GitHub Pages or any static host.

Duration: 6 weeks   Certificate: Available
